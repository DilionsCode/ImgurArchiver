# ImgurArchiver

How to use it!

```
require "Scraper.php";
// Parameter is the tag or subreddit so t/anime or r/anime_irl
$scrap = new Scraper('r/anime_irl');
$scrap->getImages();
```

Include the `Scraper.php` and put the subreddit or tag in the paramter where `anime_irl` is.
